# Utility-Tool
A Utility Tool made in C# with WinForms

## Current Features

- Runs Clean Manager to clean disk drives
- Cleans temporary files
- Cleans prefetch file
- Cleans cache memory
- Defrag an option of C:/, D:/, E:/ and F:/
- Button to retrieve current operating system
